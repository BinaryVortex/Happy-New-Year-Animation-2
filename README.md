## Demo
Open `index.html` in any modern browser to view the animation. No build tools or server required.

## Features
- Smooth CSS animations with JavaScript enhancements
- Works on desktop and mobile browsers
- Easily customizable colors, text and timing
- Small and dependency-free (pure HTML/CSS/JS)

## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/BinaryVortex/Happy-New-Year-Animation-2.git
   ```
2. Open the project folder and double-click `index.html` (or serve it via any static server).

## Usage
- Open `index.html` in your browser.
- If you want to tweak the animation, edit the relevant CSS in the stylesheet and JavaScript in `script.js` (or similarly named files).
- To change the displayed text, edit the HTML where the greeting is defined.

## Customization
- Colors: Adjust CSS variables or color rules in the main stylesheet to match your theme.
- Timing: Change animation-duration or tweak JS timers to slow down or speed up effects.
- Text: Replace the greeting text in `index.html` to personalize the message.

Example CSS variables you might find and change:
```css
:root {
  --bg-color: #000;
  --primary: #ffeb3b;
  --accent: #ff5252;
}
```
